# Program B — B0 taxonomy (starter)

**Date:** 2026-07-21  
**Status:** draft starter

## Input axes
1. Surjectivity of \(F\) over \(\mathbb{R}^n\)  
2. Regularity (polynomial / smooth / analytic)  
3. Existence of global dual fields \(X_j\) with \(DF X_j=e_j\), div-free  
4. Target package: ESS / deficiency pattern / strong CCR / CP  

## Coarse bins
| Bin | Typical outcome |
|-----|-----------------|
| Non-surjective + dual fields exist | some \(X_j\) incomplete (structural) |
| A001-like all-\(j\) incomplete | full ESS failure possible |
| Surjective Keller (if any over \(\mathbb{R}\)) | dual-flow obstruction from image fails; other issues open |
| No dual fields | out of dual-lift class |
