## 1. Mission

Classify algebraic endomorphisms arising from Keller-type (or more general local-diffeo) maps
\[
F:\mathbb{R}^n\to\mathbb{R}^n,\quad
\Phi(q,p)=(F(q),J^{-T}p),\quad
\psi\text{ Weyl lift}
\]
according to analytic realizability:

| Class | Meaning |
|-------|---------|
| **B-ESS** | All / some \(H_j=-iX_j\) essentially self-adjoint |
| **B-Def** | Deficiency index pattern \((n_+^{(j)},n_-^{(j)})\) |
| **B-CCR** | Strong CCR / dual-translation package possible? |
| **B-CP** | Unital CP extension of \(\psi\) possible? (pointer to Program C) |
