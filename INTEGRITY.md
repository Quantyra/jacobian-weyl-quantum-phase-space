# INTEGRITY — Standing non-claims boundary

This repository is a research scaffold for the Quantyra Jacobian / Weyl / quantum phase-space program (planning epic E005).

## Standing non-claims

Until expanded under an approved claim-boundary protocol with independent human approval:

1. **Jacobian Conjecture validity.** This project does not assert that any particular JC counterexample is correct. Literature intake pins the object of study; external counterexample status must be cited and may be contested, conditional, or withdrawn.
2. **No new quantum algorithms or hardware claims.** This project does not claim new quantum algorithms, complexity separations, or hardware results.
3. **No physical gate claims without criteria and proof.** This project does not claim that exotic Weyl endomorphisms are physical quantum gates (unitaries, CCR automorphisms, or symplectic-preserving maps) without an explicit reversibility / unitarity / automorphism / symplectic criterion and supporting proof or obstruction.
4. **No P-vs-NP.** This project does not claim P-vs-NP results, NP/circuit lower bounds, Frege/PHP results, or AC0 collapse results.

## Allowed public language (scaffold stage)

- Bounded theorem language, literature maps, definitions, and negative/obstruction results.
- Program structure, citation packages, and methodology notes.
- Explicit “object of study” statements that defer validity to cited external work.

## Forbidden public language (scaffold stage)

- “We prove the Jacobian Conjecture is false.”
- “This yields a new quantum gate / algorithm / speedup.”
- “Exotic endomorphisms are unitary” without criteria + evidence.
- Any P-vs-NP or circuit complexity claim derived from this lane.
